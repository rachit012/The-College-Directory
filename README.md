# The College Directory

A simple and responsive frontend application that lists colleges with search and sorting functionality.  
---

## Features

- Displays a list of colleges using mock data
- Search colleges by name 
- Sort colleges by fees (Low to High)
- Fully responsive layout
  - Desktop: 3 cards per row
  - Mobile: 1 card per row

---

## Tech Stack

- **React (Vite)**
- **Tailwind CSS**
- JavaScript
